As part of our KUET Bus Tracking System for the CSE 3200 System Development Project, we have completed the initial phase of development focused on map data processing and routing logic.

Current Progress

Collected and processed Khulna road network data from OpenStreetMap (GeoJSON format).

Constructed a graph representation (nodes and bidirectional edges) from the road data.

Implemented the A* search algorithm using a min-heap priority queue, achieving O((V + E) log V) time complexity.

Verified that the algorithm produces correct shortest path results with efficient computation.

Developed a small web-based prototype to visually demonstrate the routing functionality.

In the attached prototype:

The Khulna road network is displayed on an interactive map.

Two points can be selected as start and end locations.

The system computes and visualizes the shortest path using A*.

Distance and basic performance metrics are displayed.
